O Git é um software que funciona como um gerenciador de versões e repositório local para código para sites em desenvolvimento, ou seja, ao desenvolver um site muitas vezes são necessárias várias versões e tentativas até chegar ao produto final, para que todas estejam salvas e com backups feitos usamos esse gerenciador.

O Github é um software que funciona como um repositório online (remoto) para guardar arquivos e códigos. Bem como permite a alteração do repositório por terceiros.

Palavres chaves: Push para o repositório remoto --> Jogar os arquivos no repositório remoto
